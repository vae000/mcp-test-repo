# GitHub MCP API文档

本文档介绍GitHub MCP的主要API功能。

## 用户管理
- `get_me` - 获取当前用户信息
- `search_users` - 搜索GitHub用户

## 仓库操作
- `search_repositories` - 搜索仓库
- `create_repository` - 创建新仓库
- `list_branches` - 列出分支
- `create_branch` - 创建新分支

## 文件操作
- `get_file_contents` - 获取文件内容
- `create_or_update_file` - 创建或更新文件
- `delete_file` - 删除文件

## 问题管理
- `list_issues` - 列出问题
- `create_issue` - 创建问题
- `add_issue_comment` - 添加评论
